# acnl

Animal Crossing: New Leaf savegame toolkit

## Local usage

```sh
pnpm i
pnpm acnl <DIRECTRORY> <PARSER>
```

- Directory: Path to the folder containing `garden_plus.dat` or other savegame data.
- Parser:
  - `garden_plus`
  - `exhibition(wa)`
  - `mydsgn`
